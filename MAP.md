# 🗺 Карта уровней — Pixel Platformer

## Связи между уровнями

```
  ┌──────────┐     направо     ┌──────────┐     финиш
  │  Jungle  │ ──────────────→ │  Temple  │ ──────→ 🏁
  │ (ур. 1)  │ ←────────────── │ (ур. 2)  │
  └──────────┘     налево      └──────────┘
```

- **Вход в уровень** — зависит от откуда пришёл:
  - Обычный старт: слева (x=40, y=200, velX=0, velY=0)
  - Пришёл направо из предыдущего уровня: появляется с левого края (x=-5, сохранённая y, сохранённая скорость) и влетает в экран
  - Пришёл налево из следующего уровня: появляется с правого края (x=325, сохранённая y, сохранённая скорость) и влетает в экран
- **Переход к следующему** — выйти за правый край (x > 330)
- **Возврат на предыдущий** — выйти за левый край (x < -20)
- **Завершение игры** — коснуться флага (goal) на Храме

## Правила перехода

1. **Jungle (ур.1)**: левая граница x=0 — стена, герой не может выйти налево. Правая граница x=350.
2. **Temple (ур.2)**: левая граница x=-20 — можно вернуться в Jungle. Правая граница x=340.
3. Земля достроена до краёв границ, чтобы игрок не провалился.
4. При переходе сохраняется:
   - `coinCount` — общее количество собранных монет
   - `startY` — вертикальная позиция игрока (выходит на той же высоте)
   - `from` — имя уровня-отправителя

## Прогресс монет

- Монеты записываются в `BaseLevel.collectedCoins` (Set)
- Ключ монеты: `"Jungle-44-174"` (уровень + координаты)
- При повторном входе на уровень собранные монеты не создаются
- Сброс только при перезагрузке страницы
