# 🗺 Карта уровней — Pixel Platformer

## Связи между уровнями

```
  ┌──────────┐     направо     ┌──────────┐     финиш
  │  Jungle  │ ──────────────→ │  Temple  │ ──────→ 🏁
  │ (ур. 1)  │ ←────────────── │ (ур. 2)  │
  └──────────┘     налево      └──────────┘
```

- **Вход в уровень** — игрок появляется слева (x=40)
- **Переход к следующему** — дойти до правого края (x > 330)
- **Возврат на предыдущий** — дойти до левого края (x < -20)
- **Завершение игры** — коснуться флага (goal) на Храме

## Правила перехода

1. **Границы мира** расширены: x от -20 до 350 (против стандартных 320)
2. Земля достроена до этих границ, чтобы игрок не провалился
3. При переходе сохраняется:
   - `coinCount` — общее количество собранных монет
   - `startY` — вертикальная позиция игрока (выходит на той же высоте)
   - `from` — имя уровня-отправителя

---

## Прогресс монет

- Монеты записываются в `BaseLevel.collectedCoins` (Set)
- Ключ монеты: `"Jungle-44-174"` (уровень + координаты)
- При повторном входе на уровень собранные монеты не создаются
- Сброс только при перезагрузке страницы
