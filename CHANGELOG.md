# Changelog

All notable changes to this project are tracked here.

---

## [1.0.114] — 2026-06-27

### Исправлено
- **Jungle (левый край):** граница мира теперь x=0 (стена). Раньше было -20, игрок мог уйти влево за экран. Первый уровень — возврата нет.
- **Версия:** поднята до 1.0.114 во всех файлах (index.html, main.js, base.js).

### Добавлено
- **CHANGELOG.md** — этот файл, логи изменений между сессиями.

---

## [1.0.113] — 2026-06-27

### Исправлено
- **Jungle и Temple:** земля расширена до 350-360px, `world.setBounds` настроены на выход за правый край (переход между уровнями).
- **base.js:** `setCollideWorldBounds(true)` сохранён, границы мира задаются в каждом уровне отдельно.
- **MAP.md** — создана карта уровней.

---

## [1.0.112] — 2026-06-27

### Добавлено
- Финишная зона (goal) на Temple.
- Статический `Set collectedCoins` — монеты не появляются при возврате на уровень.
- Метод `createCoins()` в base.js.
- `checkFall()` — респавн при падении за экран (y > 280).

### Исправлено
- `staticSprite` + `group.add()` → `group.create()` (двойное добавление монет).
- `win()` передаёт `coinCount` при переходе.
- `win()` не падает при `nextScene === null` (конец игры).
- Touch-флаги сбрасываются в `init()`.
- Порядок создания: player → goal → coins (был баг с undefined).
- Версия вынесена в константу.

### Удалено
- `checkers.html` — шашки больше не в проекте.

---

## [1.0.111] — исходная версия

- Базовый платформер на Phaser 3.60.
- 2 уровня: Jungle и Temple.
- Переходы между уровнями через край экрана.
